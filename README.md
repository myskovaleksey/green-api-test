# Автотесты для GREEN-API (WhatsApp)

Тестовое задание на разработку автотестов (API + Jest + TS).

## Стек технологий
- **TypeScript** + **Jest** (Test Runner)
- **Axios** (HTTP Client)
- **Dotenv** (Configuration)

## Описание тестов
1. **getStateInstance**: Проверка авторизации инстанса перед началом тестов.
2. **sendMessage**: 
   - Успешная отправка текстового сообщения (200 OK).
   - Проверка обязательного поля `idMessage`.
   - Ошибка при отправке пустого сообщения (400 Bad Request).
3. **getChatHistory**:
   - Успешное получение истории (200 OK).
   - Проверка структуры ответа (массив, наличие обязательных полей в объектах).
   - Ошибка при некорректных параметрах запроса (400 Bad Request).

## Запуск
1. Установите зависимости: `npm install`
2. Скопируйте `.env.example` в `.env` и заполните ваши данные инстанса.
3. Запустите тесты: `npm test`
